# Moodz

**Moodz** is an emotionally aware audio companion designed to help users regulate stress, overwhelm, and low mood through personalised therapeutic sound.

Built for **Unihack 2026** by **Malong Not Found**.

## Overview
Moodz helps users find calm in difficult moments through emotionally matched audio. Instead of recommending music based only on taste, Moodz responds to emotional state using a curated therapeutic audio library and AI-assisted recommendations.

Moodz is **not a replacement for therapy or professional care**. It is a support tool for emotional regulation and relaxation.

## Features
- Personalised onboarding
- Emotion-based recommendations
- Free-form emotional search
- Curated therapeutic audio library
- Audio playback with looping
- Cross-platform support on iOS, Android, and web
- Breathing exercise

## Tech Stack
- **Frontend:** React Native, Expo
- **Backend / Auth / Database:** Supabase
- **Audio Storage:** AWS S3
- **Search:** Elasticsearch
